# <a name="jsgetownpropertynames-function"></a>JsGetOwnPropertyNames – funkce
Získá seznam všech vlastností v objektu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
STDAPI_(JsErrorCode) JsGetOwnPropertyNames(  
   _In_ JsValueRef object,  
   _Out_ JsValueRef *propertyNames  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `object`  
 Objekt, ze kterého chcete získat názvy vlastností.  
  
 `propertyNames`  
 Pole názvy vlastností.  
  
## <a name="return-value"></a>Návratová hodnota  
 Kód `JsNoError` v případě, že byla operace úspěšná, jinak kód chyby.  
  
## <a name="remarks"></a>Poznámky  
 Vyžaduje aktivní kontext skriptu.  
  
## <a name="requirements"></a>Požadavky  
 **Záhlaví:** jsrt.h
