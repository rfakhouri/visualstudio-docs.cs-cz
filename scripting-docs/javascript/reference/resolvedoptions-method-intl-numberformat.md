# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions – metoda (Intl.NumberFormat)
Vrátí objekt obsahující vlastnosti a hodnoty `NumberFormat` formátování objektu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parametry  
 `numberFormatObj`  
 `NumberFormat` Objekt, který chcete načíst informace z.  
  
## <a name="remarks"></a>Poznámky  
 Vlastnosti vráceného objektu odpovídají počítané vlastnosti `NumberFormat` objektu.
