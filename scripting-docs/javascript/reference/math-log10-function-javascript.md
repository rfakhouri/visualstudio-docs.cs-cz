# <a name="mathlog10-function-javascript"></a>Math.log10 – funkce (JavaScript)
Vrátí logaritmus o základu 10 čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
Math.log10(x)   
```  
  
## <a name="return-value"></a>Návratová hodnota  
 Pokud `x` je `NaN`, funkce vrátí hodnotu `NaN`. Pokud `x` je menší než nula (0), funkce vrátí hodnotu `NaN`.
