# <a name="void-operator-javascript"></a>void – operátor (JavaScript)
Zabraňuje výrazu vracet hodnotu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Poznámky  
 `expression` Argumentem je libovolný platný [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] výraz.  
  
 `void` Operátor vyhodnotí výraz v jeho a vrátí `undefined`. Je užitečné v situacích, kdy by měl být výraz vyhodnocen, ale nechcete, aby výsledky viditelné pro zbytek skript.
