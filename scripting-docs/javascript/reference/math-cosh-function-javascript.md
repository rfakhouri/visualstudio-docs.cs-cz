# <a name="mathcosh-function-javascript"></a>Math.cosh – funkce (JavaScript)
Vrátí hyperbolický kosinus čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.cosh(x)   
```  
  
## <a name="remarks"></a>Poznámky  
 Vrácená hodnota je stejná jako `(exp(x) + exp(-x))/2`. Pokud `x` je `NaN`, výsledkem je `NaN`.
