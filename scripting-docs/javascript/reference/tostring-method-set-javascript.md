# <a name="tostring-method-set-javascript"></a>toString – metoda (Set) (JavaScript)
Vrátí řetězcovou reprezentaci sady.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>Parametry  
 `setObj`  
 Požadováno. A `Set` objektu.  
  
## <a name="property-valuereturn-value"></a>Hodnota vlastnosti / návratová hodnota  
 Řetězcová reprezentace sady.
