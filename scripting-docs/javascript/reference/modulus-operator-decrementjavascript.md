# <a name="modulus-operator--javascript"></a>Operátor numerického zbytku (JavaScript)
Rozdělí hodnotu číselného výrazu hodnotou jiného numerického výrazu a vytvoří zbytek.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>Arguments  
 `result`  
 Všechny proměnné.  
  
 `expression1`  
 Jakýkoli číselný výraz.  
  
 `expression2`  
 Jakýkoli číselný výraz.
