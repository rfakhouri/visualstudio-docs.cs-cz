# <a name="tostring-method-weakmap-javascript"></a>toString – metoda (WeakMap) (JavaScript)
Vrátí řetězcovou reprezentaci `WeakMap` objektu.  
  
## <a name="syntax"></a>Syntaxe  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>Parametry  
 `weakmapObj`  
 Požadováno. A `WeakMap` objektu.  
  
## <a name="property-valuereturn-value"></a>Hodnota vlastnosti / návratová hodnota  
 Řetězcová reprezentace `WeakMap`.
