# <a name="mathsinh-function-javascript"></a>Math.sinh – funkce (JavaScript)
Vrátí inverzní hyperbolický sinus čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Poznámky  
 Vrácená hodnota je stejná jako `(exp(x) - exp(-x))/2`. Pokud x je `NaN`, výsledkem je `NaN`.
