# <a name="mathcos-function-javascript"></a>Math.cos – funkce (JavaScript)
Vrátí kosinus čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `radians` argument je číselného výrazu, který obsahuje úhel v radiánech.  
  
 Vrácená hodnota je kosinus číselné argument `radians`.
