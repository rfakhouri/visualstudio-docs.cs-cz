# <a name="assignment-operator--javascript"></a>Operátor přiřazení (=) (JavaScript)
Přiřazuje hodnotu proměnné.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>Parametry  
 `result`  
 Všechny proměnné.  
  
 `expression`  
 Jakýkoli číselný výraz.  
  
## <a name="remarks"></a>Poznámky  
 = Operátor chová jako dalšími operátory, aby výrazy, které obsahují jeho hodnotu. To znamená, že operátory přiřazení můžete takto řetězu: `j = k = l = 0`. V takovém případě `j`, `k`, a `l` roven nule.
