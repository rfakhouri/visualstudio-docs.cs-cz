# <a name="mathsqrt-function-javascript"></a>Math.sqrt – funkce (JavaScript)
Vrátí druhou odmocninu čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## <a name="remarks"></a>Poznámky  
 Požadované `number` argument je číselný výraz.  
  
 Pokud `number` je záporná, je vrácenou hodnotu `NaN`.
