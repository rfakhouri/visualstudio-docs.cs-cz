# <a name="razor-support"></a>Podpora Razor

Visual Studio pro Mac teď oficiálně podporuje úpravy Razor, včetně technologie IntelliSense a zvýraznění syntaxe v `.cshtml` soubory.

![syntaxe Razor úpravy v sadě visual studio pro mac](media/razor-image1.png)

## <a name="getting-started-with-razor-in-visual-studio-for-mac"></a>Začínáme s Razor v sadě Visual Studio pro Mac

Existují dvě možnosti, které je třeba zvážit při zahájení práce s jádrem Razor v sadě Visual Studio pro Mac: stránky Razor v ASP.NET Core a ASP.NET Core MVC. Další informace o obě možnosti a kurzy naleznete následující příručky:

- [Začínáme s stránky Razor v ASP.NET Core v systému macOS pomocí sady Visual Studio pro Mac](https://docs.microsoft.com/aspnet/core/tutorials/razor-pages-mac/razor-pages-start?view=aspnetcore-2.1)
- [Začínáme s ASP.NET MVC jádra a sady Visual Studio pro Mac](https://docs.microsoft.com/aspnet/core/tutorials/first-mvc-app-mac/start-mvc?view=aspnetcore-2.1)
