# <a name="razor-support"></a>Podpora Razor

Visual Studio for Mac podporuje úpravy Razor, včetně technologie IntelliSense a zvýrazňování syntaxe ve *.cshtml* soubory.

![razor editing in visual studio for mac](media/razor-image1.png)

## <a name="getting-started-with-razor-in-visual-studio-for-mac"></a>Začínáme s Razor v sadě Visual Studio pro Mac

Existují dvě možnosti, které je třeba zvážit při zahájení práce s jádrem Razor v sadě Visual Studio pro Mac: Stránky Razor v ASP.NET Core a ASP.NET Core MVC. Kurzy a další informace o obou možnostech najdete jednu z níže uvedených vodítek si:

- [Začínáme se stránkami Razor v ASP.NET Core v systému macOS pomocí sady Visual Studio pro Mac](/aspnet/core/tutorials/razor-pages-mac/razor-pages-start?view=aspnetcore-2.1)
- [Začínáme s ASP.NET Core MVC a sady Visual Studio pro Mac](/aspnet/core/tutorials/first-mvc-app-mac/start-mvc?view=aspnetcore-2.1)
