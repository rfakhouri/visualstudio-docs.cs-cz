# <a name="managing-app-resources"></a>Správa prostředků aplikace

Soubory prostředků aplikace, například bitové kopie, textové soubory a zvukových souborů jsou nezbytné pro aplikace, ale nejsou kompilovat s aplikací. Jednotlivé platformy podporované sadou Visual Studio pro Mac zpracovává tyto prostředky různými způsoby, jak je popsáno v následujících příručkách:

## <a name="xamarinforms"></a>Xamarin.Forms

Spustí kód Xamarin.Forms ve více platformách - z nichž každá má svou vlastní systému souborů a každý systém souborů určuje, jak číst a zapisovat do souborů. V Xamarin.Forms můžete spravovat prostředky aplikace pomocí nativních rozhraní API souboru na každou platformu, nebo přidávání souborů jako vložené prostředky.

* [Práce s obrázky](https://developer.xamarin.com/guides/xamarin-forms/user-interface/images/)
* [Práce se soubory]( https://developer.xamarin.com/guides/xamarin-forms/application-fundamentals/files/)
