 Některé balíčky byste měli odebrat nikdy, protože řadu dalších funkcí trvat závislosti na ně. Například byste měli odebrat nikdy uvedené v části "Základní IDE".  
  
 Některé funkce nemohou být zcela odstraněny. Například neexistuje žádný balíček, který můžete zrušit její registraci odebrat **zobrazení tříd** a jeho přidružené nabídek, možnosti a služby. **Zobrazení tříd** okno poskytuje prostředí balíčku sady Visual Studio, který poskytuje také další funkce, klíče IDE. Pokud jste chtěli odebrat **zobrazení tříd**, budete také muset odebrat **najít a nahradit**, prostředí **možnosti** stránky, **příkazové okno**a **výstup** okno.
