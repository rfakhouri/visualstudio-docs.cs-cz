# <a name="microsoft-help-viewer-installation"></a>Instalace aplikace Microsoft Help Viewer

Několik produktů můžete zobrazit obsah nápovědy v Microsoft Help Viewer, včetně sady Visual Studio a SQL Server.

Aplikaci Help Viewer je volitelná instalace součástí sady Visual Studio. K jeho instalaci pomocí instalačního programu sady Visual Studio, postupujte podle těchto kroků:

1. Otevřete **instalační program sady Visual Studio** z nabídky Start nebo pokud máte Visual Studio otevřete, můžete zvolit **nástroje** > **stažení nástrojů a funkcí** otevřete vizuál Instalační program sady Studio.

1. Zvolte **jednotlivé komponenty** kartu a potom vyberte **aplikace Help Viewer** pod **kódu nástroje** oddílu.

   ![Komponenta prohlížeče nápovědy instalačního programu VS](media/help_viewer_vs_installer.png)

1. Zvolte **změnit** tlačítko pro spuštění instalace aplikace Microsoft Help Viewer.

Dalším způsobem, jak snadno instalovat aplikace Microsoft Help Viewer je prostřednictvím **Snadné spuštění** pole:

1. Zadejte nebo vložte **aplikaci help viewer** v **Snadné spuštění** pole v záhlaví okna sady Visual Studio.

   ![Rychlé spuštění pole](media/help_viewer_quick_launch.png)

1. Vyberte výsledek názvem **aplikace Help Viewer (jednotlivých komponent)**.

1. V dialogovém okně, které se otevře, zvolte **nainstalovat** tlačítko.

   ![Tlačítko Instalovat](media/help_viewer_install.png)
