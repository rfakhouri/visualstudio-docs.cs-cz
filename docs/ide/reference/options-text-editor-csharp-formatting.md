# <a name="options-text-editor-c-formatting"></a>Možnosti, textový editor, C#, formátování

Použití **formátování** možnosti stránky lze nastavit možnosti formátování kódu v editoru kódu. Chcete-li získat přístup k této stránce Možnosti, zvolte **nástroje** > **možnosti**a potom vyberte **textového editoru** > **C#**  >  **Kódu stylu** > **formátování**.

> [!NOTE]
> Dialogová okna a příkazy nabídek, které vidíte, se mohou lišit od těch popsaných v nápovědě v závislosti na aktivních nastaveních nebo edici. Chcete-li změnit nastavení, zvolte **nastavení importu a exportu** na **nástroje** nabídky. Další informace najdete v tématu [přizpůsobení prostředí Visual Studio IDE](../../ide/personalizing-the-visual-studio-ide.md).

## <a name="general-settings"></a>Obecná nastavení

Obecné nastavení ovlivnit jak editoru kódu se týká možnosti formátování kódu.

## <a name="uielement-list"></a>Seznam prvků uživatelského rozhraní

|Popisek|Popis|
|-----------|-----------------|
|**Automaticky formátu při psaní**|Pokud není vybraná, **formátu příkaz na;** a **Formát bloku na}** možnosti jsou zakázány.|
|**Automaticky formátu příkaz na;**|Při výběru naformátuje příkazy při dokončení podle možnosti formátování vybrané pro editor.|
|**Automaticky Formát bloku na}**|Při výběru formáty kódu bloky podle možnosti formátování vybrané pro editor, jakmile dokončíte blok kódu.|
|**Formát automaticky při návratu**|Při výběru formáty text při **Enter** stisknutí, přizpůsobit možnosti formátování vybrané pro editor.|
|**Automaticky formát na vložení**|Při výběru formáty text, který se vloží do editoru podle možnosti formátování vybrané pro editor.|

## <a name="preview-window"></a>Okno náhledu

**Odsazení**, **nové řádky**, **mezery**, a **zabalení** možnosti podokna zobrazení okno náhledu. Okno náhledu ukazuje účinek jednotlivých možností. Chcete-li použít okno náhledu, vyberte možnost formátování. Okno náhledu ukazuje příklad vybrané možnosti. Pokud změníte nastavení tak, že vyberete přepínač nebo zaškrtněte políčko, aktualizace okno náhledu zobrazíte účinek se nové nastavení.

## <a name="remarks"></a>Poznámky

Odsazení možnosti na **karty** stránky pro každý jazyk pouze určit, kde editoru kódu umístí kurzor po stisknutí klávesy **Enter** na konci řádku. Odsazení možnosti v části **formátování** použít, když kód formátována automaticky, například při vložení kódu do souboru při **automaticky formátovat vkládané** je vybrána a kdy bloku formátovaného je zadán ručně.
