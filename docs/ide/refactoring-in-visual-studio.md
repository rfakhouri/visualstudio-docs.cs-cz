# <a name="refactoring-in-visual-studio"></a>Refaktoring v sadě Visual Studio
Refaktoring je proces změnu kódu, aby bylo snazší spravovat, pochopit a rozšíření, ale bez změny chování aplikace. Některé běžné *refaktoring* zahrnují:  
  
1.  dlouhé funkce rozdělení na menší části  
  
2.  Přejmenování proměnné, typy, funkce nebo parametry  
  
3.  Změna pořadí parametrů  
  
4.  odebrání nebo změna uspořádání pomocí direktiv  
  
 Protože logiku refaktoring úzce souvisí syntaxe jazyka, každý služba jazyka v sadě Visual Studio nabízí vlastní refaktoringu funkce. Další informace najdete na následujících stránkách:
