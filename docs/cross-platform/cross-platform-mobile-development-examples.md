# <a name="cross-platform-mobile-development-examples"></a>Příklady mobilní vývoj pro různé platformy
Některé šablony nainstalované ve Visual C++ pro vývoj mobilních řešení pro různé platformy generování dokončení příklady, které se můžete dozvědět se od. Kromě toho webu Windows Dev Center má několik příklad aplikace, které si můžete stáhnout a vyzkoušet v sadě Visual Studio.  
  
-   [Ukázkové aplikace Android Hello jni](https://code.msdn.microsoft.com/hello-jni-Android-790ab73d)  
  
     Tato ukázka je port aplikace hello jni Android NDK. Ukázka ukazuje začátku do konce Java nativní rozhraní aplikace "Hello World". Načte řetězec z nativní metoda implementována ve sdílené knihovně a potom zobrazí v aplikaci.  
  
-   [Ukázkové aplikace Android Hello gl2](https://code.msdn.microsoft.com/hello-gl2-Android-3b61896c)  
  
     Tato ukázka je port aplikace hello gl2 Android NDK. Ukázka ukazuje aplikace Java nativní rozhraní Android OpenGL začátku do konce. Vykreslí trojúhelník pomocí shaderu OpenGL ES 2.0 rozhraní API.  
  
-   [Ukázkové aplikace pro Android plazmy rastrového obrázku](https://code.msdn.microsoft.com/Bitmap-Plasma-Android-77ae296a)  
  
     Tato ukázka je port Android NDK rastrový obrázek plazmy aplikace. Ukázka ukazuje aplikace Java nativní rozhraní Android OpenGL ES 2.0 začátku do konce. Ukazuje přímá manipulace Android rastrový obrázek pixelů vyrovnávací paměti pro generování plazmy vliv.  
  
-   [Ukázka knihovny TwoLibs Android](https://code.msdn.microsoft.com/TwoLibs-Android-Library-6396e5c4)  
  
     Tato ukázka je port Android NDK TwoLibs vzorku. Používá dynamicky načíst sdílené knihovny i statické C++ Android nativní knihovny, který implementuje metodu s názvem z aplikace Java nativní rozhraní. Tato ukázka je to dobrý výchozí bod pro vývojáře, abyste pochopili, jak používat sdílené knihovny statické nebo dynamické k sestavení aplikace platformy Android JNI začátku do konce s Visual Studiem 2015.  
  
-   [Ukázkové aplikace pro Android Pot čaj](https://code.msdn.microsoft.com/Tea-Pot-Android-Application-e7c05d73)  
  
     Tato ukázka je port Android NDK TeaPot aplikace. Ukázka ukazuje aplikace Java nativní rozhraní Android OpenGL ES 2.0 začátku do konce.  
  
-   [Ukázkové aplikace MoreTeaPots Android](https://code.msdn.microsoft.com/MoreTeaPots-Android-a9bd8549)  
  
     Tato ukázka je port Android NDK MoreTeaPots aplikace. Ukázka ukazuje aplikace Java nativní rozhraní Android OpenGL začátku do konce.  
  
-   [Ukázka testů libstdcpp Android knihovny](https://code.msdn.microsoft.com/test-libstdcpp-Android-00b548f5)  
  
     Tato ukázka je port testu Android NDK-libstdc ++ ukázce speciálně pro použití s Visual Studio 2015. Tato ukázka je to dobrý výchozí bod pro vývojáře, abyste pochopili, jak používat standardní knihovny.  
  
 Chcete-li spustit některý z příkladů v sadě Visual Studio, stáhněte si soubor zip a otevřete **vlastnosti** stránky stažený soubor v Průzkumníku. Vyberte **Odblokovat** tlačítko a potom vyberte **OK**. Extrahujte obsah souboru zip do vhodného umístění, potom otevřete složku C++ v ukázce extrahované a otevřete soubor řešení.  
  
 Chcete-li sestavit ukázku, stiskněte klávesu F7 nebo na řádku nabídek zvolte **sestavení**, **sestavit řešení**.
