```  
void write_message(  
   _In_ LPCTSTR _Format,  
   ...  
);

   marker_importance _Importance,  
   _In_ LPCTSTR _Format,  
   ...
```  

 `_Importance`  
 Úroveň důležitosti.  
  
 `_Category`  
 Category.Importance úroveň.
