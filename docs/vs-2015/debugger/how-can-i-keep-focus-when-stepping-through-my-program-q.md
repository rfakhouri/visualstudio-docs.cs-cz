# <a name="how-can-i-keep-focus-when-stepping-through-my-program"></a>Jak mohu zachovat fokus při procházení programem?
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [jak mohu zachovat fokus při krokování přes tento Program?](https://docs.microsoft.com/visualstudio/debugger/how-can-i-keep-focus-when-stepping-through-my-program-q).  
  
Popis  
 Program má problém Aktivace okna. Krokování pomocí programu v ladicím programu dochází ke kolizím s Moje schopnost reprodukujte problém, protože program spouštím ztrácí fokus. Existuje způsob, abyste tomu předešli?  
  
## <a name="solution"></a>Řešení  
 Pokud máte druhý počítač, použijte vzdálené ladění. Při spuštění ladicího programu na hostiteli můžete provozovat programu na vzdáleném počítači. Další informace najdete v tématu [postupy: výběr vzdáleného počítače](http://msdn.microsoft.com/en-us/4332ba8e-2f0b-4f62-b96a-e762b9f3c3ba).
