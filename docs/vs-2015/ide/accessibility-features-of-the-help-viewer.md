# <a name="accessibility-features-of-the-help-viewer"></a>Funkce usnadnění programu Help Viewer
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Microsoft usiluje o usnadnění svých produktů a služeb pro každodenní použití. Toto téma obsahuje informace o funkcích, produktech a službách, které pomáhají zpřístupnit Microsoft 2.2 Prohlížeč nápovědy pro osoby s různými schopnostmi.  
  
## <a name="keyboard-access"></a>Použití klávesnice  
 Všechny funkce aplikace Help Viewer můžete přistupovat pomocí klávesnice. Další informace najdete v tématu [klávesové zkratky (Help Viewer)](../ide/shortcut-keys-help-viewer.md).  
  
## <a name="font-size"></a>Velikost písma  
 Můžete změnit velikost písma, ve které téma text se zobrazí v okně dokumentu. Další informace najdete v tématu [přizpůsobit aplikaci Help Viewer](../ide/customize-the-help-viewer.md).  
  
## <a name="window-size"></a>Velikost okna  
 Šířku okna navigace nebo dokumentu můžete změnit pouhým ukázáním na oddělovač mezi dvěma okny. Když se ukazatel změní na oboustrannou šipku, pomocí primárního tlačítka myši přetáhnout rozdělovací příčku směrem doprava nebo doleva.  
  
## <a name="help-viewer-position"></a>Pozice Prohlížeč nápovědy  
 Aplikace Help Viewer můžete přemístit přetažením záhlaví na jiné místo.  
  
## <a name="internet-explorer-options"></a>Možnosti aplikace Internet Explorer  
 Přístupnější aplikaci Help Viewer můžete provést tak, že změníte možnosti aplikace Internet Explorer, které se týkají způsobu zobrazování webových stránek. Další informace najdete v tématu [přizpůsobit aplikaci Help Viewer](../ide/customize-the-help-viewer.md).
