# <a name="ca0501"></a>CA0501

Nelze číst výstup sestavy.

Tato situace nastane, když výstup sestav se nedá zapisovat z jednoho z následujících důvodů:

-   Disk je plný

-   Žádná oprávnění k zápisu do adresáře

-   Adresář neexistuje
