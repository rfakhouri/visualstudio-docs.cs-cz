# <a name="c28225"></a>C28225
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28225](https://docs.microsoft.com/visualstudio/code-quality/c28225).  
  
upozornění C28225: nalezen správný počet požadovaných parametrů v poznámce  
  
 Toto upozornění signalizuje, že poznámka s názvem se používá s nesprávným počtem parametrů. To by neměl být možné v případě, že standardní makra jsou používány pro poznámky; Toto upozornění se neočekává se zobrazí v typické použití.
