# <a name="c6515"></a>C6515
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C6515 – neplatná Poznámka: \<name > vlastnost může být použita pouze na hodnotách typu ukazatel  
  
 Toto upozornění signalizuje, že byla použita vlastnost pro použití na ukazatelích na typech bez ukazatele typu. Seznam vlastností poznámky, naleznete v tématu [vlastnosti poznámky](http://msdn.microsoft.com/en-us/f77b4370-6bda-4294-bd2a-e7d0df182a3d).  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```cpp  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char pc,  size_t c)  
{  
    // code ...  
}  
```  
  
 Chcete-li opravit toto upozornění, použijte následující kód:  
  
```  
#include <sal.h>  
  
void f(_Readable_bytes_(c) char * pc,  size_t c)  
{  
    // code ...  
}  
```
