# <a name="c28226"></a>C28226
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Nejnovější verzi tohoto tématu můžete najít v [C28226](https://docs.microsoft.com/visualstudio/code-quality/c28226).  
  
Upozornění C28226: Poznámka nemůže být také PrimOp (v aktuální deklaraci)  
  
 Toto upozornění signalizuje, že je deklarované jako PrimOp Poznámka s názvem a také se dřív deklaroval jako normální poznámky. To by neměl být možné v případě, že standardní makra jsou používány pro poznámky; Toto upozornění se má zobrazit při běžném používání.
