# <a name="ca0505"></a>CA0505

Vlastnost nepoužívané bude ignorován, protože vlastnost nahrazující je definována.

Upozornění CA505 se vygeneruje, když zadaný zastaralé i nahrazující vlastností analýzy kódu obsahují hodnoty v projektu nebo sestavení konfiguračního souboru. Chcete-li tento problém vyřešit, odeberte vlastnost zastaralé.
