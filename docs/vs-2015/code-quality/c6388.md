# <a name="c6388"></a>C6388
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

upozornění C6388: \<argument > nemusí být \<hodnota >: to nedrží se specifikací pro funkci \<název funkce >: Řádky: x, y  
  
 Toto upozornění signalizuje, že Neočekávaná hodnota se používá v zadaném kontextu. Obvykle je hlášena pro hodnoty předávané jako argumenty funkci, která to neočekává.  
  
## <a name="example"></a>Příklad  
 Následující kód jazyka C++ generuje toto upozornění, protože DoSomething očekává hodnotu null, ale mohla být předána potenciálně nenulová hodnota:  
  
```cpp  
  
#include <string.h>  
#include <malloc.h>  
#include <sal.h>  
  
void DoSomething( _Pre_ _Null_ void* pReserved );  
  
void f()  
{  
    void* p = malloc( 10 );  
    DoSomething( p );  // Warning C6388  
    // code...  
    free(p);  
}  
```  
  
 Chcete-li opravit toto upozornění, použijte následující ukázkový kód:  
  
```cpp  
  
#include <string.h>  
#include <malloc.h>  
#include <sal.h>  
  
void DoSomething( _Pre_ _Null_ void* pReserved );  
void f()  
{  
    void* p = malloc( 10 );  
    if (!p)  
    {  
        DoSomething( p );    
    }  
    else  
    {  
        // code...  
        free(p);  
    }  
}  
```  
  
 Všimněte si, že použití malloc a free má mnoho nástrah v podobě nevracení paměti a výjimek. Chcete-li zabránit těmto druhům nevrácené a výjimku problémů zcela, použijte mechanismy, které jsou k dispozici ve C++ knihovny STL (Standard Template). Patří mezi ně [shared_ptr](https://msdn.microsoft.com/library/1469fc51-c658-43f1-886c-f4530dd84860), [unique_ptr](https://msdn.microsoft.com/library/acdf046b-831e-4a4a-83aa-6d4ee467db9a), a [vektoru](https://msdn.microsoft.com/library/c1431ad8-c0b6-4dbb-89c4-5f651e432d7f). Další informace najdete v tématu [inteligentní ukazatele](https://msdn.microsoft.com/library/909ef870-904c-49b6-b8cd-e9d0b7dc9435) a [standardní knihovny C++](https://msdn.microsoft.com/library/a37d3ba3-58af-47c7-9ee2-441ccd7b77ee).
