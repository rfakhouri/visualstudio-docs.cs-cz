# <a name="idiasymbolgetiscvtcil"></a>IDiaSymbol::get_isCVTCIL
Získá příznak označující, zda modul byl převeden do modulu Common mezilehlá jazyk souboru CIL () na nativní modul.  
  
## <a name="syntax"></a>Syntaxe  
  
```C++  
HRESULT get_isCVTCIL(  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pFlag`  
 [out] Vrátí `TRUE` Pokud modul byla převedená z soubor CIL do nativního kódu; jinak vrátí `FALSE`.  
  
## <a name="return-value"></a>Návratová hodnota  
 V případě úspěchu vrátí `S_OK`, jinak vrátí `S_FALSE` nebo chybový kód.  
  
> [!NOTE]
>  Vrácená hodnota `S_FALSE` znamená vlastnost není k dispozici pro symbol.  
  
## <a name="remarks"></a>Poznámky  
 Tato vlastnost je k dispozici z `SymTagCompilandDetails` typu symbolu (viz [compilanddetails –](../../debugger/debug-interface-access/compilanddetails.md).
