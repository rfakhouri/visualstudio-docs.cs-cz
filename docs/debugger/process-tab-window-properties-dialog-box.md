# <a name="process-tab-window-properties-dialog-box"></a>Karta Proces, dialogové okno vlastnosti okna
Použití **proces** kartě k identifikaci procesu vybrané okno. Chcete-li zobrazit [dialogové okno Vlastnosti okna](../debugger/window-properties-dialog-box.md), přesunout fokus na [zobrazení oken](../debugger/windows-view.md) okno. Vyberte libovolný uzel okno ve stromové struktuře, a potom vyberte **vlastnosti** z **zobrazení** nabídky.  
  
 Následující nastavení jsou k dispozici na **proces** karty:  
  
|Položka|Popis|  
|-----------|-----------------|  
|**ID procesu**|ID procesu, který vlastní vlákno, které vytvořili toto okno. Vyberte tuto hodnotu zobrazíte vlastnosti tohoto procesu.|  
|**ID podprocesu**|ID podprocesu, který vytvořili toto okno. Vyberte tuto hodnotu zobrazíte vlastnosti tohoto podprocesu.|
