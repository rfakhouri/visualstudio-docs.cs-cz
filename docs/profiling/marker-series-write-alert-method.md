# <a name="markerserieswritealert-method"></a>marker_series::write_alert – metoda

## <a name="syntax"></a>Syntaxe  
  
```  
void write_alert(  
   _In_ LPCTSTR _Format,  
   ...  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `_Format`
