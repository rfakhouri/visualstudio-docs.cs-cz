# <a name="what39s-new-in-the-xml-editor"></a>Co&#39;s nové v editoru XML
Následující funkce jsou nové v sadě Visual Studio 2010.  
  
## <a name="xslt-intellisense"></a>XSLT IntelliSense  
 Funkci XSLT IntelliSense umožňuje automatické dokončování atribut sadu názvů; režimy šablon a názvů; a názvy parametrů pro zadaný režim nebo zadané s názvem šablony. Další informace najdete v tématu [editoru XML IntelliSense funkce](../xml-tools/xml-editor-intellisense-features.md).  
  
## <a name="xslt-profiler"></a>Profiler XSLT  
 Profileru XSLT vytvoří podrobné XSLT výkonu sestavy, které vám pomůže měřit, hodnocení a cíle problémy související s výkonem v kódu XSLT. Profileru XSLT je také užitečné pomocné parametry pro XSL a XSLT optimalizace na list stylu. Další informace najdete v tématu [návod: XSLT profileru](../xml-tools/walkthrough-xslt-profiler.md).  
  
## <a name="xslt-hierarchy"></a>Hierarchie XSLT  
 Nástroj XSLT Hierarchy můžete přidat zarážky ve součástí šablony stylů a integrované šablony pravidla. Další informace najdete v tématu [návod: použití hierarchie XSLT](../xml-tools/walkthrough-using-xslt-hierarchy.md).
