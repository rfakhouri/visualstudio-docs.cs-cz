# <a name="c26471-noreinterpretcastfromvoidptr"></a>C26471 NO_REINTERPRET_CAST_FROM_VOID_PTR

Nepoužívejte `reinterpret_cast`. Přetypování z void * může používat `static_cast`. Zobrazit [Type.1 pokyny pro jádro C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-reinterpretcast).
