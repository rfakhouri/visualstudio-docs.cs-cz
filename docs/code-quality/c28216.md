# <a name="c28216"></a>C28216
upozornění C28216: _Check_return\_ poznámky platí jenom pro následující po podmínky pro parametr konkrétní funkce.  
  
 `_Check_return_` Poznámky byla použita v kontextu jiné než `__post`; může být nutné `__post` (nebo `__drv_out`) modifikátor, nebo může být umístěn nesprávně.
