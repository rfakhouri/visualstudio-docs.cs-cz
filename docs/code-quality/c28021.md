# <a name="c28021"></a>C28021
upozornění C28021: Parametr \<param > s poznámkou \<anno > musí být ukazatel

 Toto upozornění je hlášeno, pokud objekt poznámkou není typ ukazatele. Tato poznámka nemůže být použita s `void` nebo celočíselné typy.
