# <a name="ca0059"></a>CA0059
Přepínač příkazového řádku došlo k chybě.  
  
 K této chybě dojde, pokud vyžaduje příkazového řádku přepínače chybí, například  
  
 / out nebo zadáním parametru není zadán. K této chybě dojde, pokud byl neplatný přepínač, jako je například hodnota chybný časového limitu.
