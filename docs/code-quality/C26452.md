# <a name="arithmetic-overflow-left-shift-count-is-negative-or-greater-than-or-equal-to-the-operand-size-which-is-undefined-behavior"></a>Aritmetické přetečení: Počet levých posunů je záporný nebo větší než nebo roven velikosti operandu, což je nedefinované chování

 Toto upozornění signalizuje, počet posunů je záporný nebo větší než nebo roven počtu bitů operand je posunuta, což vede k nedefinovanému chování.
Poznámka: C4293 je podobné kontrolu v kompilátoru C++ společnosti Microsoft.

## <a name="example"></a>Příklad

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return (hi << 32) | lo; // C26252 here
}
```
 Chcete-li opravit toto upozornění, použijte následující kód:

```cpp
unsigned __int64 combine(unsigned lo, unsigned hi)
{
  return ((unsigned __int64)hi << 32) | lo; // OK
}
```
