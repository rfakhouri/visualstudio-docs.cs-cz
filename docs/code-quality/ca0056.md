# <a name="ca0056"></a>CA0056
Soubor projektu má nesprávnou verzi nástroje pro analýzu.

 K tomuto upozornění dochází, pokud FxCopCmd.exe je vyvolána ručně pomocí souboru projektu, který obsahuje verzi nástroje analýzy nesprávný kód.
