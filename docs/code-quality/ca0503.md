# <a name="ca0503"></a>CA0503
Vlastnost je zastaralá. Nahrazující vlastnost

 Upozornění CA0503 se vygeneruje, když vlastnost analysis nepoužívané kód obsahuje hodnotu v konfiguračním souboru sestavení nebo projektu. K vyřešení toto upozornění, použijte zadanou vlastnost, která nahrazuje ho.
