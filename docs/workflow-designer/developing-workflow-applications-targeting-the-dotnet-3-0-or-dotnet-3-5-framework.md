# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>Vývoj aplikací pracovní postup zacílený na rozhraní .NET 3.0 nebo 3.5 rozhraní .NET Framework
[!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] poskytuje návrháře pracovních postupů Windows pro vytváření [!INCLUDE[wf](../workflow-designer/includes/wf_md.md)] aplikacích, které cílí [!INCLUDE[netfx40_long](../workflow-designer/includes/netfx40_long_md.md)]. Návrhář používá známé [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] uživatelské rozhraní pro grafické vytváření a ladění aplikací pracovního postupu.

 [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] také poskytuje přístup k návrháře starší verze pracovního postupu, který lze použít k cíli [!INCLUDE[netfx35_long](../workflow-designer/includes/netfx35_long_md.md)] nebo [!INCLUDE[vstecwinfx](../workflow-designer/includes/vstecwinfx_md.md)]. Tato část obsahuje pokyny k použití této starší verzi návrháře.
