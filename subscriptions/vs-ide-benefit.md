# <a name="using-the-visual-studio-2017-benefit"></a>Používání benefitu Visual Studio 2017
Jako odběratel Visual Studio máte přístup k prostředí Visual Studio IDE.  Verze bude záviset na úrovni předplatného.  Výhodou se zobrazí jako dlaždici v nástroji kategorii na [výhody](https://my.visualstudio.com/benefits) stránce portálu odběratele.  

(Visual Studio Enterprise se použije v tomto příkladu.). 

Chcete-li stáhnout integrovaného vývojového prostředí:
1. Klikněte **Stáhnout** odkaz v levém dolním dlaždice výhody sady Visual Studio. Také si můžete stáhnout z [stáhne](https://my.visualstudio.com) stránky. 

![Visual Studio Enterprise dlaždice](_img\vs-ide-experience\vs-ide-tile.png)

2.  Klikněte na odkaz "Stáhnout" v dolní části dlaždici výhody.  Můžete budete přesměrováni na stránku stahování výsledky pro Visual Studio Enterprise, kde budete mít příležitost stáhnout prostředí IDE a získat kód product key aktivovat. Kliknutím na modrý se také mohou prohlásit kód product key **získat klíč** tlačítko nebo deklarací klíč později na [kódy Product Key](https://my.visualstudio.com/productkeys) stránky.
3.  Na kartě Podrobnosti stránky s výsledky stáhnout:
- Vyberte vaší požadované architektury (x86, x64)
- Vyberte váš preferovaný jazyk. 
4.  Kliknutím na tlačítko Stáhnout zelená.
![Podrobnosti o sadě Visual Studio Enterprise stahování](_img\vs-ide-experience\vs-ide-download-details-cropped.png)
5.  Pokud chcete instalovat hned, klikněte na tlačítko "Spustit".
6.  V dialogovém okně Řízení uživatelských účtů klikněte na tlačítko Ano..
7.  Klikněte na tlačítko **pokračovat** přijmout licenční podmínky.
8.  Když se spustí instalační program, zvolte si požadované úlohy, jednotlivých součástí a jazykových sad výběrem na odpovídající kartě.  
![Přizpůsobení sady Visual Studio Enterprise](_img\vs-ide-experience\vs-ide-customize-install-cropped.png)
9.  Klikněte na tlačítko **nainstalovat**. 
10. Po dokončení instalace budete mít možnost upravte instalaci nebo spuštění sady Visual Studio.  Klikněte na tlačítko **spusťte** začít pracovat.  
![Spusťte Visual Studio Enterprise](_img\vs-ide-experience\vs-ide-launch-cropped.png)
11. Přihlaste se pomocí účtu spojené s vaším předplatným Visual Studio. 
12. Vyberte upřednostňovaný motivu a klikněte na **spuštění sady Visual Studio**.
13. Úvodní stránka se zobrazí.  Jste připravení začít s použitím sady Visual Studio!
![Začít používat Visual Studio Enterprise](_img\vs-ide-experience\vs-ide-start-cropped.png)
