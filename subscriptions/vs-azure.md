## <a name="activation-steps"></a>Postup aktivace

1.  Přihlaste se k [ https://my.visualstudio.com/benefits ](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

2.  Vyhledejte dlaždici Azure v části nástroje na výhody a klikněte na tlačítko **aktivovat** odkaz v dolní části dlaždici výhody.

    ![Azure dlaždice](_img\vs-azure\vs-azure-tile.png)

3.  Zadejte požadované informace k vytvoření vašeho předplatného Azure, klikněte na zaškrtávací políčko přijmout podmínky a klikněte na tlačítko **zaregistrovat**:

    ![Registrace Azure](_img\vs-azure\vs-azure-sign-up-cropped.png)

4.  Pokud váš účet je připraven, budete mít možnost Vybrat plán podpory Azure (prodává samostatně) nebo začít pracovat s předplatným Azure.

    ![Probíhá příprava Azure](_img\vs-azure\vs-azure-getting-ready-cropped.png)

5.  Načte řídicí panel Azure a na první použití, budete mít možnost prohlídka.  Prohlídku se skládá z řady tipy obrazovky, které bude odkazovat na tom, jak můžete provádět různé úlohy a hledání příslušných informací.  Dokončení nebo ukončování prohlídku se zobrazí řídicí panel Azure.

6.  Záložky [portálu Azure](https://portal.azure.com) pro usnadnění přístupu v budoucnu.
![Řídicí panel Azure](_img\vs-azure\vs-azure-dashboard-cropped.png)

## <a name="faq"></a>Nejčastější dotazy

### <a name="q-how-do-i-submit-a-technical-support-incident-from-within-the-azure-portal"></a>Otázka: Jak mohu odeslat technické podpory incidentu z portálu pro Azure?
Odpověď: odesílání na základě incidentu podpory z portálu Azure je proces třech krocích.
1. Aktivovat výhody vaší se na technickou podporu a získat ID vašeho přístupu ID smlouvy.
2. Propojení vaší smlouvě o podpoře k předplatnému Azure.
3. Na základě incidentu podpory pro odeslání.

Navštivte [se na technickou podporu](vs-tech-support.md) dokumentace úplné podrobnosti.

### <a name="q-ive-received-a-message-stating-that-my-access-to-azure-will-end-soon--what-can-i-do-to-continue-using-azure"></a>Otázka: jste zobrazila zpráva, že brzy skončí Moje přístup k Azure.  Jak pokud chcete pokračovat v používání Azure?

A: Tento e-mail je jste obdrželi, protože používáte Azure benefit spojené s vaším předplatným Visual Studio, a že má předplatného vypršela platnost, nebo byla odebrána.  Chcete-li pokračovat v používání Azure, máte několik možností:

**Doporučená akce:**

Chcete-li pokračovat v používání vašich platebních Visual Studio Azure a slevy, proveďte jednu z následujících kroků:

Pokud někdo v váš tým nebo organizace zakoupí odběry vaším jménem, obraťte se na správce vaší organizace Visual Studio a žádostí předplatné sady Visual Studio, které vás opravňuje k použití Azure měsíční kredit množství, které potřebujete.   Požádejte správce o nový odběr přiřadit stejné ID uživatele, kterému byla přiřazena předchozí předplatné, takže nemusíte si dělat starosti o přenos vašeho předplatného Azure.


Pokud přímo můžete zakoupit předplatné, proveďte jednu z následujících:
1.  Zakoupit roční Visual Studio [Enterprise](https://marketplace.visualstudio.com/items?itemName=ms.vs-enterprise-annual) nebo [Professional](https://marketplace.visualstudio.com/items?itemName=ms.vs-professional-annual) cloudové předplatné přes Visual Studio Marketplace.
2.  Využít výhod 50 % nebo vyšší slevu obnovením vaším aktuálním předplatným prostřednictvím [Microsoft Store](https://www.microsoft.com/en-us/search/result.aspx?q=visual+studio+subscription).

Porovnání všechny [odběry a jejich výhody](https://visualstudio.microsoft.com/vs/pricing/).

Pokud máte jiný aktivní předplatné v sadě Visual Studio **stejné** předplatné úrovni pod účtem Microsoft, můžete se propojit ho k této Azure využívat tak, že přidáte [alternativní účet](manage-vs-subscriptions.md#linking-my-subscription-to-existing-vsts-or-azure-subscriptions) v sadě Visual Studio předplatné portálu. Pokud to nepomůže, můžete také zkusit [přenosu vaší Azure benefit](/azure/billing/billing-subscription-transfer/) na aktivní předplatné sady Visual Studio.

Pokud máte jiný aktivní předplatné v sadě Visual Studio **různých** předplatné úroveň a pod účtem Microsoft, obraťte se na [podporu Azure](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade) o pomoc.

Pokud vaše předplatné má limitu útraty a provádět žádnou akci před jeho platnost vyprší, předplatná Azure, budou zakázány vašeho předplatného Azure. Chcete-li ji převést na nabídku průběžných plateb a ho nadále používat, přidejte platebního nástroje a [limit útraty odebrat](/azure/billing/billing-spending-limit/).

Pokud nechcete pokračovat toto předplatné Visual Studio Azure, můžete převést vaše prostředky, jako je například virtuální počítače a databáze do jiného předplatného Azure a [zrušit](http://azure.microsoft.com/support/cancel-account/) toto předplatné kdykoli.

Pokud se domníváte, tento e-mail jste obdrželi omylem, obraťte se na [Azure zákaznickou podporu](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).
